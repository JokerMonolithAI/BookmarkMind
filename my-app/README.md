# BookmarkMind 智能书签管理平台

## 当前开发阶段
用户认证和管理系统（基于 Firebase）

## 技术栈
- Next.js 14 (App Router)
- Firebase Authentication（用户认证）
- Firebase Realtime Database（实时数据库）
- Tailwind CSS（样式框架）
- Shadcn/ui（UI组件库）
